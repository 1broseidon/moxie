MOXIE

Chat agent service

v0.1.8

Connect Telegram and Slack to AI coding agents. Send a message from your phone, get a response from Claude, Codex, Gemini, or any configured backend.

$brew install 1broseidon/tap/moxie
Get StartedView Source

Features

06 Modules
001

Multi-Backend

Dispatch to Claude, Codex, Gemini, Pi, or any agent CLI. Switch backends mid-conversation with a single command.

002

Always On

Runs as a persistent service with job recovery. Messages are never lost, even across restarts.

003

Telegram & Slack

Native transports for both platforms. Send a message from your phone, get a response from any AI agent.

004

Schedules

One-shot timers, cron schedules, and delayed dispatches. Automate recurring agent tasks without leaving the chat.

005

Subagents

Delegate focused work to a different backend. Results are synthesized back into the parent conversation.

006

Powered by oneagent

Backend config, output parsing, and thread management handled by oneagent. Adding a new agent is a JSON config edit.

Quick Start

# Install and configure
brew install 1broseidon/tap/moxie
moxie init

# Start the service
moxie serve

# Now in Telegram or Slack:
# "explain the auth module"        → dispatches to Claude
# /model codex                     → switches backend
# "fix the login bug"              → dispatches to Codex
# /think high                      → enables extended thinking
